Semantic HTML and Screen Readers

In this module, we're going to look at an aspect of Semantic coding that affects interactivity. An HTML button looks like a button by default. It can be styled to looked more interactive. Semantically, it is the way to code a clickable element in HTML. In this example, we'll see that we can make any item clickable in the sense that we can trigger an action based on a click. The problem is that a screen reader won't find a name unless it's a button. We'll see how to check look for this name in the Chrome browser.
