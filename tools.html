Web Accessibility Guidelines

Web Content Accessbility Guidelines (WCAG) is a list of "Success Criteria" that can help a web page to meet the requirements for a web page to be easy to use, even with a disability. These requirements are broken down into categories representing 5 dimensions of accessibility. The list below names the dimensions and provides Some examples of success criteria in each dimension. The Conformance category provides measure that can be use to specify how well a web page conforms to the guidelines.

  1. Perceivable

  2. Operable
  3. Understandable
  4. Robust
  5. Conformance
