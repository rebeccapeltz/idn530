A blind student from Berkeley couldn't read their website with a screen reader because they were using images without alt tags.

Both SEO and Screenreaders benefit from using Semantic HTML. Using semantic HTML adds meaning to the structure of your document and this helps automatic reading of the page by web crawlers and screen readers to provide accurate information about the content. Automation relies on structures than can be easily parsed.

What's Next

We've learned a number of ways to measure website accessbility and make websites more accessible. These techniques are as important for training websites as they are for commercial websites. Students are content consumers and come to a website with the need to be able to digest content easily.

It can be difficult to balance providing an engaging website using IFRAMEs and interesting styles, colors and animations with ensuring accessbility. Screen readers help millions to learn from websites. There is much to be done to provide accessible web pages and much to be done to make it easier to make web pages accessible.
